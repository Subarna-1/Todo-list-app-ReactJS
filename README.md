# Todo List App.
![React Todo App](./banner.png1)

**Run: [Execute](https://wc-react-todo-app.netlify.app/)**

## Requirement:
- ReactJs, HTML, CSS

## Getting Started
After getting the starter files, you need to go the file directory and run

```shell
npm install
```

and after that start the dev server.

```shell
npm start
```
